=== OpenPDC Elasticsearch ===
Contributors: Yard Internet
Requires at least: 4.8
Tested up to: 5.1
Requires PHP: 7.0
Stable tag: 1.1.2

Plugin which adds Elasticsearch (via ElasticPress) capabilities to the PDC base concept to your project.
Requires OpenWebConcept OpenPDC Base plugin.
